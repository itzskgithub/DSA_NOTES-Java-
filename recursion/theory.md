🔑 The Golden Rule (memorize this)

Recursive calls go DOWN first.
Printing happens while coming BACK UP.


Resursion is all about :

        1. Base case : check for the base case first the where the function is going to be stop 

        2. Recursive work : It is the part where the function calls itself to perform recursion till the base csse.

        3. Self work : We hvae to do something from ourside to execute the process.  


        Note: Recursive work and Sefl work is decided according to the condition that which should be done first.
